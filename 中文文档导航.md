## 🎓 核心概念速记卡

### 条件 (Condition)
```
一个需要回答的问题
= 预言机 + 问题ID + 结果数量
```

### 位置 (Position)
```
你在某个结果上的权益
= 抵押代币 + 结果组合
= ERC1155 代币
```

### 分割 (Split)
```
抵押品 → 多个位置代币
锁定资产，铸造代币
```

### 合并 (Merge)
```
多个位置代币 → 抵押品
销毁代币，释放资产
```

### 赎回 (Redeem)
```
正确的位置代币 → 抵押品
根据支付向量计算
```
